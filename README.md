SignTP v1.0 <br>
========== <br>
 <br>
## Usage <br>
Create a sign with the format below <br>
   -  [WORLD]
   - \<worldname> <br>
 The remaining 2 lines can be ANYTHING YOU WANT. <br>

##Upcoming Features <br>

Coordinate signs <br>
      Coordinate signs should be in this format <br>
   - [COORDS]<br>
   - \<x position> <br>
   - \<y position> <br>
   - \<z position> <br>
